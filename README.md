# KS Language VS Code Extension

**KS Language** is a programming language extension for VS Code with custom commands, syntax highlighting, and integrated command menu.  
It is **inspired by, but separate from `.hsx`**, and focuses on cross-device compatibility, including mobile devices.

---

## Features

- Custom commands: Open, Rename, Export, Delete, Hide, Unhide, Run, Check KS files
- Syntax highlighting for `.ks` files
- Custom file icon for `.ks` files
- Compatible with desktop VS Code and mobile Python environments (Pythonista, Pydroid, Termux)
- Examples included

---

## Installation

1. Download the extension package (`.vsix`) or clone the repository.
2. Install the extension via VS Code:  
   `Extensions -> Install from VSIX...`
3. Open or create `.ks` files and start coding!

---

## Mobile Compatibility

- Extensions in the `mobile` folder are designed for mobile environments.
- Additional extensions may be added daily.  
- Users can submit multiple extensions for mobile testing.

---

## Testing Policy

- **Standard Testing Time:** 1 hour per submitted extension.
- **Partial Success (≥79% functionality but bugs present):** 6 hours testing.
- **Failure:** Moved to `mobile_failed`.
- **Success:** Moved to `mobile_succeeded`.

---

## Notes

- `.ks` is **not `.hsx`**, it’s a separate language implementation.
- Extensions submitted for mobile testing may vary from expectations due to environment limitations.
- Contributions and submissions are welcome via the repository.
